Exercico:


Claro! Que tal criar um projeto onde você exibe uma lista de filmes ou séries ao invés de atletas? Você pode usar um objeto semelhante ao que foi utilizado na aula, mas com propriedades como título, descrição e link para a imagem do filme ou série.

Aqui estão algumas sugestões para deixar o exercício mais interessante:

Adicionar um campo de gênero: Inclua um gênero para cada filme, como "Ação", "Comédia", "Drama", etc. E depois, você pode filtrar os resultados por gênero.

Implementar uma funcionalidade de busca: Adicione um campo de entrada onde o usuário pode digitar o nome de um filme e, ao clicar no botão "Pesquisar", apenas os filmes que correspondem ao que foi digitado aparecem na tela.

Estilizar a página: Use CSS para deixar a apresentação dos filmes mais bonita, como adicionar bordas, mudar cores de fundo e fontes.

Isso vai te ajudar a praticar o que aprendeu na aula e ainda te dá a chance de ser criativo! Boa sorte! 🚀✨
